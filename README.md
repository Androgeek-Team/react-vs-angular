# React.js vs AngularJs

    $ git clone git@github.com:Androgeek-Team/react-vs-angular.git
    $ cd react-vs-angular
    $ npm install
    $ grunt
    $ node server.js

Open [http://localhost:3000/](http://localhost:3000/)
